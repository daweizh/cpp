## step1

书写程序主入口函数框架

~~~cpp
int main() {

    return 0;
}
~~~

在**Dev Cpp**中，按**F9**编译，按**F11**编译执行。

## step2

增加输入输出功能

~~~cpp
#include <iostream>
    
int main() {
    std::cout << "Hello, World!" << std::endl;
    return 0;
}
~~~

## step3

用另一种方法说Hello, World!

~~~cpp
#include <iostream>
    
int main(){
    printf("Hello, World!");
    return 0;
}
~~~

## step4

简化`std`前缀

~~~cpp
#include <iostream>
using namespace std;
    
int main(){
    cout << "Hello, World!" << endl;
    return 0;
}
~~~
