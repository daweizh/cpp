# Q04.01 判断数正负

总时间限制: 1000ms 内存限制: 65536kB

## 题目描述

给定一个整数N，判断其正负。

## 输入

一个整数N(-10<sup>9</sup> <= N <= 10<sup>9</sup>)

## 输出   

如果N > 0, 输出positive;   
如果N = 0, 输出zero;   
如果N < 0, 输出negative。

## 样例输入

    1

## 样例输出

    positive
