# Q05.13 乘方计算

总时间限制: 1000ms 内存限制: 65536kB

## 题目描述
   
给出一个整数a和一个正整数n，求乘方a<sup>n</sup>。

## 输入
   
一行，包含两个整数a和n。-1000000 <= a <= 1000000，1 <= n <= 10000。

## 输出
   
一个整数，即乘方结果。题目保证最终结果的绝对值不超过1000000。

## 样例输入


    2 3

## 样例输出


    8
